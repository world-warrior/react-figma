---

React Figma supports different color formats.

* By names: `red`, `green`, etc. according to [CSS Color Module level 4](https://drafts.csswg.org/css-color/#named-colors)
* Hex
  + `#RGB`
  + `#RGBA`
  + `#RRGGBB`
  + `#RRGGBBAA`
* Rgb: `rgb(10, 20, 30)`, `rgba(10, 20, 30, .3)`
