# Hello, world!

MDX is an authorable format that lets you seamlessly write JSX in your Markdown documents.

It allows for displaying something unique from your design system. E.g., colors:

<Row>
    <Color value="#6A737D" />
    <Color value="#0366D6" />
    <Color value="#28A745" />
    <Color value="#6F42C1" />
    <Color value="#FFD33D" />
    <Color value="#F66A0A" />
    <Color value="#D73A49" />
</Row>
